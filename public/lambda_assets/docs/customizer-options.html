Customizer Options

List of all the customizer options available

Site

Name Description
Site Skin Sets the current skin used to style the site.
Layout Type Sets the sites body layout.
options : Normal, Boxed

Logo

Name Description
Logo Text Add your logo text here.
Logo Image Upload an image to use as the sites logo.
Logo Transparent Image Upload an image to use as the sites logo when page has a transparent header.

Header

Name Description
Style Choose a layout for your headers menu & logo
options : Logo Left - Menu Right, Logo Right - Menu Left, Logo Right - Menu Below, Logo Left - Menu Below, Logo Center - Menu Below, Logo Left - Menu Sidebar
Menu Align Set the alignment of the dropdown menu.
options : Left, Right
Width Set the width of the header container.
options : Normal, Full Width
Sticky Make the navigation stick to the top of the page.
options : On, Off
Top Bar Adds a top bar to the top of your page above the main header.
options : On, Off
Text Capitalization Sets the case of the text inside your header.
options : Force Uppercase, Force Lowercase, Off
Navbar Scroll Change Point Point in pixels after the page scrolls that will trigger the menu to change height.
Hover Menu Choose between menu that will open when you click or hover (desktop only option since mobile devices will always use touch)
options : Click, Hover
Hover Menu Delay Delay in seconds before the hover menu closes after moving mouse off the menu.
Hover Menu Fade Delay Delay of the Fade In/Fade Out animation .

Upper Footer

Name Description
Upper Footer Columns Select how many columns the upper footer will consist of.
options : 0, 1, 2, 3, 4
Upper Footer Top Padding Sets the amount of padding to add to the top of the upper footer.
Upper Footer Bottom Padding Sets the amount of padding to add to the bottom of the upper footer.

Footer

Name Description
Footer Columns Select how many columns the footer will consist of.
options : 0, 1, 2, 3, 4
Footer Top Padding Sets the amount of padding to add to the top of the footer.
Footer Bottom Padding Sets the amount of padding to add to the bottom of the footer.
Back to top button Show or hide the back to top button that appears when you scroll down the page.
options : Enable, Disable
Back to top shape Set the shape of the back to top button.
options : Square, Circle

Sub Footer

Name Description
Sub Footer Columns Select how many columns the footer will consist of.
options : 0, 1, 2, 3, 4
